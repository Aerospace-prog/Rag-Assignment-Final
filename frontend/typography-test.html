Font Sizes

text-xs (0.75rem / 12px): Caption and label text
text-sm (0.875rem / 14px): Secondary text and small content
text-base (1rem / 16px): Body text and main content
text-lg (1.125rem / 18px): Emphasized text
text-xl (1.25rem / 20px): Small headings
text-2xl (1.5rem / 24px): Section headings
text-3xl (1.875rem / 30px): Page headings

Font Weights

font-light (300): Light weight text for subtle emphasis
font-regular (400): Regular weight for body text
font-medium (500): Medium weight for buttons and labels
font-semibold (600): Semibold for subheadings
font-bold (700): Bold for headings and emphasis

Line Heights

leading-tight (1.25): Tight line height for headings and compact text. This is useful when you want to save vertical space while maintaining readability.
leading-normal (1.5): Normal line height for body text and general content. This provides comfortable reading experience for most text content.
leading-relaxed (1.75): Relaxed line height for improved readability. This is ideal for longer paragraphs and content that needs extra breathing room.

Letter Spacing

tracking-tight (-0.025em): Tight letter spacing for large headings
tracking-normal (0): Normal letter spacing for body text
tracking-wide (0.025em): Wide spacing for uppercase labels
tracking-wider (0.05em): Wider spacing for section headers

Font Families

font-primary (Inter): The quick brown fox jumps over the lazy dog. 0123456789
font-mono (Fira Code): const greeting = "Hello, World!"; // Code sample
